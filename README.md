# My-Movie-List
同济大学软件学院2019年网页开发课程第二次作业展示

1551719 张文喆

## 1. 功能

### 基础功能

1. 读取JSON文件并渲染数据
2. 电影列表展示
3. 分页

### 附加功能

1. 按多种类型搜索
2. 电影详情页
3. 排行榜（按类型分榜/总榜）
4. 无法显示的海报图片自动替换
5. 可爱的favicon，见标签页左上角
